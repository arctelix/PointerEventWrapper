bind(element, action, callback)

Convenience method for binding event listeners to an element.

Simply specify the action or actions to bind and the actual bound events wil be determined by what the browser supports automatically.

Parameters:
Name Type Description
element HTMLElement

The element to bind the events
action string

Space separated string of event actions.
callback function

The callback for the events

Bind all down and up events:
